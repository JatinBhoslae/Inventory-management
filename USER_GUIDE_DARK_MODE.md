## Changing Your System Theme

If you select "System" mode, here's how to change your computer's theme:

### Windows 10/11
1. Open **Settings**
2. Go to **Personalization** ‚Üí **Colors**
3. Under "Choose your color", select:
   - **Light** for light mode
   - **Dark** for dark mode
   - **Custom** to set different themes for apps and system

### macOS
1. Open **System Preferences**
2. Click **General**
3. Under "Appearance", select:
   - **Light** for light mode
   - **Dark** for dark mode
   - **Auto** to switch based on time of day

### Linux (Ubuntu/GNOME)
1. Open **Settings**
2. Go to **Appearance**
3. Select your preferred theme

## Troubleshooting

### Theme Not Changing?

**Problem**: Clicked the theme but nothing happened

**Solutions**:
1. Try refreshing the page (F5)
2. Clear your browser cache
3. Make sure JavaScript is enabled
4. Try a different browser

### Theme Resets After Closing Browser?

**Problem**: Theme goes back to default

**Solutions**:
1. Check if your browser allows localStorage
2. Make sure you're not in Incognito/Private mode
3. Check browser privacy settings

### System Mode Not Working?

**Problem**: System mode doesn't follow OS theme

**Solutions**:
1. Make sure you selected "System" option
2. Check your OS theme is actually set
3. Try refreshing the page
4. Update your browser to the latest version

## Frequently Asked Questions

### Q: Does dark mode work on mobile?
**A**: Yes! Dark mode works on all devices including phones and tablets.

### Q: Will my theme choice sync across devices?
**A**: No, theme preference is saved per device/browser. You'll need to set it on each device.

### Q: Does dark mode save battery?
**A**: Yes, on devices with OLED/AMOLED screens, dark mode can save battery.

### Q: Can I customize the colors?
**A**: Not currently, but this may be added in a future update.

### Q: Is there a keyboard shortcut?
**A**: Not currently, but you can use Tab to navigate to the theme toggle and Enter to open it.
